# Customer Churn Analysis

This project analyzes customer churn using Python to identify the key factors that influence customer retention. The dataset was cleaned, explored, and visualized using various exploratory data analysis (EDA) techniques to uncover meaningful business insights.

## Project Objectives

* Understand customer churn patterns.
* Clean and preprocess the dataset.
* Perform exploratory data analysis (EDA).
* Identify the factors associated with customer churn.
* Present insights through clear and informative visualizations.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

* Data Loading
* Data Understanding
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Key Insights & Conclusion

## Key Insights

* Customers with month-to-month contracts have the highest churn rate.
* Customers with shorter tenure are more likely to churn.
* Higher monthly charges are associated with increased churn.
* Long-term customers are more likely to remain with the company.
* Contract type and tenure are among the strongest indicators of customer churn.

This project demonstrates practical skills in data cleaning, exploratory data analysis, and data visualization using Python, making it suitable for a beginner data analytics portfolio.
